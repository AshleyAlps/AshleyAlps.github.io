Changing the Box Model, Display & Positioning*

  • The default box-sizing value is content-box.
    • Say your content is 300 x 200, but once you add up your padding and border values, the *actual* size may be 362 x 248. This can make larger websites difficult to control & style.
  • With border-box, the height & width you set doesn't change.
    • Helpful when needing to keep an element within its parent container.
    • Can be applied to individual rulesets, or the whole page with * wildcard.
  • A document with no CSS styling will render L->R, T->B, in the same order as they exist in the document.
  • Properties for how a browser positions elements:
    • position: display: z-index: float: clear:
  • Block-level elements create a block the full width of its parent element, and prevent other elements from occupying the same horizontal space.
  • The default position can be changed by setting its position: property.
Property Value Description
box-sizing: content-box; Width & height settings apply to the content, padding & margin values add onto the total size.
border-box; Width & height settings apply to the whole element. Content shrinks to accomodate padding & margin values.
position: static; The default setting.
relative; Positions an element relative to its default position (left). Must be accompanied by a value (px, em, %) assigned to top, bottom, left, or right.
abolute; All other elements on the page will ignore it and act like it's not present on the page. Will be positioned relative to its closest positioned parent element. Offset properties can be used to further position the element.
fixed; Same as abolute, but element will remain fixed to its position regardless of user scrolling.
sticky; Similar to abolute & fixed, except it remains in the document flow. Stays in position until user scrolls, then it "sticks" to its position.
z-index: +/- integer Controls how far back/forward an element should appear when elements overlap. Value is any +/- integer; default value is 0. *Does not work on static elements*
display: inline; Can share horizontal space with another element. Cannot be altered in size with width or height. Elements with this default value: <em>, <strong>, <a>, <span>, etc.
block; Only allows this element in the horizontal space. Elements with this default value: <h1>-<h6>, <p>, <div>, <footer>, etc.
inline-block; Can share horizontal space with another element. Can be altered in size with width or height.
float: left; right; Used to move an element as far L/R as possible within the container. Must have a widthvalue or it will take up the whole horizontal space. Helpful for wrapping text around an image. *Note: Moving elements L/R is better suited for tools like CSS Grid & Flexbox.
clear: left; right; both; none; Specifies how elements should behave when they bump into each other on the page. Value determines what side *will not* touch any other element within the container.
